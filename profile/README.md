## Why This Exists

Most project management tools like Trello, Asana, and ClickUp are heavy web apps that struggle on older hardware or slow networks. They consume hundreds of megabytes of RAM, require constant internet, and bombard you with animations and telemetry.

**ZenKit To Do** solves one specific problem: the need for a fast, offline-capable, and truly minimal kanban board that runs instantly on any PC — from a modern laptop to a 10-year-old netbook. ZenKit To Do is a Trello alternative that prioritizes responsiveness over features. No background trackers, no forced cloud sync, no Electron bloat.

If you are tired of waiting for boards to load or fighting with UI lag, ZenKit To Do is for you. This Trello alternative respects your time and hardware.

## Quick Start with ZenKit To Do

1. **Get ZenKit To Do** — Download the portable `.exe` for Windows or the `.html` file for any browser.
2. **Launch** — Double-click `ZenKit_To_Do.exe` or open the HTML file. The interface appears in under 0.5 seconds.
3. **Create a board** — Click “+ Board”, name it “Work” or “Personal”.
4. **Add lists** — Default lists: To Do, In Progress, Done. Edit or delete them freely in ZenKit To Do.
5. **Add cards** — Type a task title, press Enter. Drag cards between columns.
6. **Edit or delete** — Right-click any card. ZenKit To Do keeps your data in localStorage.
7. **Work normally** — ZenKit To Do saves automatically. Close anytime, reopen, and continue.

## Understanding ZenKit To Do Core Components

ZenKit To Do is not a clone of Trello. It is a **reimagined Trello alternative** built for speed:

- **Board manager** — create multiple boards, switch instantly, delete unused boards
- **Column engine** — unlimited vertical columns, each with custom titles
- **Card system** — each card stores title, description (optional), and position
- **Drag-and-drop** — reorder cards inside same column or move between columns
- **Local storage** — all data stays on your machine. No cloud, no servers
- **Portable mode** — copy one folder or one .exe to any USB drive
- **Keyboard-first** — Esc to cancel, Enter to save, Del to delete card in ZenKit To Do

No backend, no database, no dependencies. ZenKit To Do is a pure frontend Trello alternative.
